💰 FinQuest – Gamified Financial Education Platform

FinQuest is an interactive web platform that teaches financial literacy through games, lessons, and simulations. Instead of reading long theoretical material, users learn by playing, making financial education engaging, practical, and memorable.
The platform combines interactive learning modules with gamification, allowing users to earn points, complete lessons, play financial simulations, and track their progress.

🚀 Problem Statement

Financial literacy is a crucial life skill, yet many students and young adults struggle to understand:
Budgeting and money management
Investing and compound growth
Debt and credit management
Taxes and financial planning
Traditional financial education methods are often boring, theoretical, or inaccessible.
FinQuest addresses this gap by transforming financial education into an interactive learning experience.

🎯 Solution

FinQuest uses gamification and interactive simulations to teach finance concepts in an engaging way.
Users can:

📚 Learn financial concepts through short lessons
🎮 Play interactive finance games
🏆 Earn points and rewards
📊 Track learning progress
🛒 Spend points in the avatar shop

This approach encourages active learning rather than passive reading.

🧩 Key Features
📚 Interactive Lessons

Users can access bite-sized lessons on important financial topics such as:
Budgeting
Investing
Saving
Credit & debt management
Taxes
Financial planning

🎮 Financial Learning Games

A timed quiz with questions about:
Investing

📈 Investment Simulator

Players invest virtual money across different assets:
Stocks
Crypto
Bonds
Fixed deposits

A decision-based game where users make financial choices through life stages:

Student life
First job

This helps users understand real-world financial decision making.

📊 How It Works

1️⃣ User signs up or logs in
2️⃣ User accesses the dashboard
3️⃣ User completes lessons or games
4️⃣ Points are awarded for progress
5️⃣ Points can be spent in the shop
6️⃣ Progress is tracked through the leaderboard and profile
